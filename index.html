Q2:返回三者中最小两数的平方和
def two_of_three(i, j, k):
“””Return mm + nn, where m and n are the two smallest members of the
positive numbers i, j, and k.

>>> two_of_three(1, 2, 3)
5
>>> two_of_three(5, 3, 1)
10
>>> two_of_three(10, 2, 8)
68
>>> two_of_three(5, 5, 5)
50
"""
return i*i+j*j+k*k-max(i,j,k)*max(i,j,k)

解开lock前的题目比较有意思

>>> def welcome():
...         print('Go')
...         return 'hello'
>>> def cal():
...         print('Bears')
...         return 'world'
>>> welcome()

答案：
(line 1)? GO
(line 2)? ‘hello’

Q: Which of the following is NOT true?
Choose the number of the correct choice:
1) It is generally good practice to release code with assertion statements left in
   发布代码时保留断言语句通常是良好的编程习惯
2) Debugging is not a substitute for testing
   调试不能替代测试
3) Code that returns a wrong answer instead of crashing is generally better as it at least works fine
   代码返回错误答案而非直接崩溃一般更好，因为它至少能正常运行
4) Testing is very important to ensure robust code
   测试对于保障代码健壮性非常重要
5) It is generally bad practice to release code with debugging print statements left in
   发布代码时遗留调试打印语句是不良编程习惯

答案为3
静默错误（返回错误结果却不崩溃）比程序直接崩溃危害更大：崩溃会立刻暴露问题，而错误输出会悄悄产生错误数据，难以察觉，从而造成更严重的后果。该说法完全错误。
